## 📌 Project Overview
This repository contains the data, visualizations, and theoretical framework for the study: **"Why a Dominant L3 May Be Easier to Suppress than a Developing L4 during L5 Acquisition"**.

### The Core Hypothesis: The Proficiency-Inhibition Paradox
We propose that inhibitory control operates differently across the multilingual repertoire:
- **High Proficiency (L1-L3):** High Automaticity $\rightarrow$ Efficient Global Inhibition.
- **Intermediate Proficiency (L4):** Active Monitoring $\rightarrow$ Inhibitory Leakage.

## 📊 Key Findings
- **L4 (Russian) Interference:** 91% of total intrusions.
- **L3 (English) Interference:** 9% of total intrusions.
- **Conclusion:** There is a "U-shaped" relationship where dominant languages are better suppressed than developing ones.
